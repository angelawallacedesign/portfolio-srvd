# Standalone Privacy Hero

This directory contains a dependency-free copy of the privacy hero.

- `component.html`: markup only
- `component.css`: all required styles and local tokens
- `component.js`: vanilla JavaScript animation initializer
- `index.html`: local demo using the split files
- `paste-ready.html`: single-file snippet with inline CSS and JS

Use `paste-ready.html` when you want to paste one HTML block into another project without copying separate assets.
