# Standalone Components

These components and modules are packaged for independent use within the portfolio. Each package includes its own markup, styles, and JavaScript unless otherwise noted.

## Pattern Inspector

Path: `inspector-refactor/pattern-inspector-integration/`

- Use `pattern-inspector.html`, `pattern-inspector.css`, and `pattern-inspector.js` together.
- Use `pattern-inspector-loader.js` to mount the module inside an existing HTML slot.
- Open `demo.html` to preview and test the standalone integration package.
- Refer to `pattern-inspector-feature-spec.md` or `pattern-inspector-feature-spec.docx` for feature behavior, architecture, implementation, and future Design Inspector recommendations.
- The `pass-01-semantic-layout/`, `pass-02-annotations/`, and `pass-03-pattern-navigation/` directories preserve approved implementation checkpoints and are not the integration source.

## Previous Inspector Prototype

Path: `archive/inspector/`
